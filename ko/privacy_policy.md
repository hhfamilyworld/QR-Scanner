---

## 1. 수집하는 정보

앱은 외부 서버에 이용자 데이터를 저장하거나 전송하지 않습니다. 아래 정보는 이용자 기기 내에서만 처리됩니다.

**기기 내 저장 정보 (서버 전송 없음)**

| 항목 | 용도 |
|------|------|
| 스캔한 QR코드·바코드 내용 | 스캔 결과 표시 및 히스토리 저장 |
| 앱 설정값 (테마, 언어 등) | 앱 환경설정 유지 |

**특정 기능 사용 시 처리되는 정보**

| 항목 | 처리 시점 | 용도 |
|------|-----------|------|
| 연락처 정보 | QR 연락처 저장 기능 사용 시 | 기기 주소록에 저장 (서버 전송 없음) |
| 위치 정보 (Wi-Fi용) | Wi-Fi QR 연결 기능 사용 시 | Wi-Fi 네트워크 연결 (저장·전송 없음) |
| 광고 식별자 (GAID) | 광고 동의 시 | 맞춤형 광고 제공 (Google AdMob) |

## 2. 앱 권한 사용 목적

| 권한 | 필수/선택 | 목적 |
|------|-----------|------|
| 카메라 | 선택 | QR코드·바코드 실시간 스캔 |
| 사진·갤러리 | 선택 | 이미지에서 QR코드 인식 |
| 연락처 | 선택 | vCard QR코드 스캔 후 주소록 저장 |
| 위치 (NEARBY_WIFI_DEVICES) | 선택 | Wi-Fi QR코드로 네트워크 연결 |

선택 권한을 거부해도 해당 기능만 제한되며, 앱의 기본 기능은 계속 이용할 수 있습니다.

---

## 3. 정보 보유 기간

| 항목 | 보유 기간 |
|------|-----------|
| 스캔 히스토리 | 이용자가 직접 삭제하거나 앱을 삭제할 때까지 |
| 앱 설정값 | 앱 삭제 시 자동 삭제 |
| 광고 식별자 | 광고 동의 철회 또는 앱 삭제 시 |

스캔 히스토리는 이용자 기기의 로컬 데이터베이스(SQLite)에만 저장되며, 개발자 서버에는 어떠한 스캔 데이터도 저장되지 않습니다.

---

## 4. 제3자 서비스

앱은 광고 제공을 위해 아래 서비스를 사용합니다.

**Google AdMob**

- 처리 주체: Google LLC
- 수집 항목: 광고 식별자 (이용자 광고 동의 시에만 처리)
- 방침 링크: https://policies.google.com/privacy

EEA(유럽경제지역) 및 UK 이용자는 앱 최초 실행 시 Google UMP(User Messaging Platform)를 통해 광고 개인화 동의 여부를 선택할 수 있습니다.

## 5. GDPR 안내 (EEA · UK 이용자)

EU 일반 개인정보 보호법(GDPR) 및 UK GDPR을 준수합니다.

- **동의 철회**: 앱 설정 → 광고 개인화 메뉴에서 언제든지 변경 가능
- **데이터 삭제**: 앱 삭제 시 기기 내 모든 데이터 삭제
- **열람·정정·삭제 요청**: 아래 문의처로 연락

| 처리 활동 | 법적 근거 |
|-----------|-----------|
| 기기 내 스캔 히스토리 | 정당한 이익 (핵심 기능 제공) |
| Wi-Fi QR 연결 | 정당한 이익 (이용자 요청 기능) |
| 맞춤형 광고 | 동의 (UMP 폼을 통해 수집) |
| 비맞춤형 광고 | 정당한 이익 |
